# 圣经祷告词应用 - 项目需求文档

## 项目概述
- **项目名称**: bible-prayer-app
- **类型**: Web 应用 (MVP)
- **核心功能**: AI 根据用户心情生成个性化祷告词 + 每日圣经金句
- **目标用户**: 基督徒、需要进行灵修祷告的用户

## 功能列表

### 1. 首页 (Home) ✅
- 每日祷告词（15个预设，每天轮换）
- 每日圣经段落（30个章节，每天轮换）
- AI 祷告生成入口
- 快捷操作（记录、设置）

### 2. 祷告生成 (Prayer Generator) ✅
- 8种心情选择
- AI 生成个性化祷告词（预设）
- 保存到历史记录

### 3. 祷告记录 (History) ✅
- 查看历史祷告记录
- 删除记录

### 4. 设置 (Settings) ✅
- 每日提醒时间设置（UI）
- 深色模式切换

### 5. 提醒功能 ⚠️
- UI已完成，后端推送待实现

## 技术实现
- 前端: Next.js + TypeScript + Tailwind CSS
- 状态管理: React useState
- 数据存储: localStorage (MVP阶段)
- AI生成: 模拟实现（预设祷告词，可后续接入DeepSeek API）

## 页面结构
```
/           - 首页（每日祷告词 + 圣经段落）
/prayer     - 祷告生成页
/history    - 历史记录页
/settings   - 设置页
```

- 实现真正的每日提醒推送
